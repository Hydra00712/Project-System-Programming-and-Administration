Usage Examples:
Create a Task:
./todo create "Title" "2024-06-01" "Description" "Location"

Update a Task:
./todo update 1 "New Title" "2024-06-02" "New Description" "New Location" true


Delete a Task:
./todo delete 1

Show a Task:
./todo list

Search Tasks by Title:
./todo search "Title"

Default (List Today's Tasks):
./todo
